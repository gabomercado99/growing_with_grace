## Color Usage Guide

### Primary Colors

- **Teal (#7dd3c0)**: Navbar, sidebar active states, primary buttons, borders
- **Light Teal (#9dd4d4)**: Gradients, hover states

### Accent Colors

- **Orange (#f4a261)**: Call-to-action buttons, "Learn More" links, secondary actions
- **Red (#d62828)**: Logout buttons, warnings, alerts, red flags
- **Green (#52b788)**: Checkmarks, confirmation states, positive indicators

### Neutral Colors

- **Dark Text (#333333)**: Headings and main content
- **Light Text (#666666)**: Descriptions and secondary content
- **Light Background (#f8f9fa)**: Page backgrounds, card backgrounds
- **Border (#e0e0e0)**: Card borders, dividers

## Key User Flows

### Flow 1: First-Time Visitor

1. Land on Home page
2. Browse 6 feature cards
3. Click "Get Started"
4. Redirected to Login
5. Create account (or demo login with test/test)
6. Access Portal Dashboard

### Flow 2: Parent Checking Milestones

1. Login to Portal
2. Click "Milestone Calendar"
3. Find age range for child
4. Review expected milestones
5. Check "When to Be Concerned" box
6. Navigate to Red Flags if concerned

### Flow 3: Parent Seeking Resources

1. Login to Portal
2. Browse Blog posts
3. Read research articles
4. Check Video Interventions
5. Post Q&A if has questions
6. Browse Recommended Products

### Flow 4: Shopping

1. Login to Portal
2. Go to Recommended Products
3. Review 6 product categories
4. Click "Shop on Amazon"
5. Purchase from Amazon storefront

## Responsive Behavior

### Desktop (1024px+)

- Full sidebar + content layout
- 6-column feature grids
- Full-width forms
- Horizontal navigation

### Tablet (768-1024px)

- Collapsible sidebar (hamburger menu)
- 3-column grids
- Optimized touch targets
- Reduced padding

### Mobile (<768px)

- Stack layout (sidebar below content)
- Single column content
- Touch-optimized buttons
- Simplified forms

## Animation & Interaction

### Hover Effects

- Card elevation (+8px)
- Color transitions (0.3s)
- Border color changes
- Opacity changes (0.8)

### Loading States

- Button disabled state
- Spinner/skeleton screens
- Fade-in animations

### Transitions

- All: 0.3s ease-in-out
- Form inputs: 0.2s
- Page navigation: 0.5s fade
